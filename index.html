What CSS Is Used For

CSS can be used for simple things like changing the color's of HTML tags.
It can also change values of a tag, as
well as change a value when a certain
action happens! For example, a:hover is
used in CSS to change an anchor tag value when you hover over a link!

You can do advanced things in CSS like make animations and, with enough time, make complex objects! Maybe even 3D looking shapes!

I made this small project possible with the help of powerful CSS!
